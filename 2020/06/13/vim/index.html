Vim 古老編輯器的簡明使用方法


如果你有使用 git 的話
你一定見過 vim
像是 merge 別人的分支
或是 commit 的時候

基本操作

以下介紹:

如何使用 vim 開啟文檔

使用 vim 開啟文檔的方法非常簡單,
只需要在 CLI 上輸入以下指令即可:

vim /<path>/<file>

for instance:

打開所在目錄下 file.txt 文檔

vim file.txt

模式切換與編輯文件

一開始進入 vim 編輯器的時候為普通模式
在任何模式下按esc皆可以退至普通模式

在普通模式按下i可以進入插入模式
此模式可以開始進行文檔上的編輯

在普通模式下按下 : 可以進入指令模式
此模式可以退出文檔

編輯器離開方式

離開前必須進入指令模式
相關指令如下:

  • q : 離開編輯器
  • q! : 不存檔強制離開編輯器
  • w : 存檔
  • wq : 存檔離開編輯器
