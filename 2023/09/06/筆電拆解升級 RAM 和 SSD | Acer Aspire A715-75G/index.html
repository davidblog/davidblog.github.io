筆電拆解升級 RAM 和 SSD | Acer Aspire A715-75G

發表於 2023-09-06 | 分類於 生活札記 | |

舊筆電偶爾開機會遇上 No Bootable Device 的錯誤訊息
或是玩遊戲會藍屏
上網查了一些資訊可能和硬體有關
特別是 SSD 的部分

因為筆電已經過保了
想說就來手動升級
這樣不但可以修筆電
還可以順便提升性能
完全就是一石二鳥

第一次自行拆機升級也上網做了很多功課
就把這個過程紀錄在部落格裡吧

  • 購買及準備零件
  • 製作 USB 開機碟
  • 拆機更換零件
  • 使用 USB 開機碟重新安裝作業系統
  • 分割及合併硬碟

購買及準備零件

主要會用到的東西有：

物品功能注意事項
USB用來製作開機碟要先確認製作開機磁碟所需要的容量
可以在搜尋欄點選修復磁碟機查看
螺絲起子用來拆機記得要選長柄的，我一開始買短柄的拆不動
翹片拆機的輔助工具呵呵，如果沒有用指甲也可以，但很辛苦。
RAM提升記憶體筆電的尺寸(SODIMM)和桌機是不一樣的，需要留意，另外時脈選擇可以參考原本的記憶體，因為有些主機板可能會有支援上的限制，這樣就算買了較高的時脈也無法享受到其性能。
SSD提升硬碟容量這邊要注意的是 SSD 的種類和尺寸

補充

SSD 的種類其實還滿多的，
但最重要的就是確認其介面和尺寸。

這邊列一下 SSD 介面主要規格的資訊：

介面描述插槽尺寸速度主要用途
SATA傳統的硬盤驅動器介面，較慢2.5 吋 或 M.2最高約 6 Gbps桌面和筆記型電腦，成本較低，適合存儲和一般任務
mSATA迷你 SATA 介面，用於特定筆記型電腦mSATA最高約 6 Gbps過去的筆記型電腦，現在較少使用
PCIe使用 PCIe 插槽，提供高速數據傳輸PCIe 插槽速度取決於規格（4.0 / 5.0 等）高性能桌面電腦，工作站，遊戲，虛擬化，專業應用等
M.2小型、高性能 SSD 介面，支持 SATA 和 NVMeM.2 插槽取決於支援的協議桌面和筆記型電腦，高性能要求，專業工作，遊戲等

製作 USB 開機碟

我的作業系統是 Windows 11
在 Windows 搜尋欄輸入修復磁碟機，然後點選開啟
接下來就是插入 USB
並按照指引說明操作即可

話說我一開始上網查
普遍的資訊是製作 USB 開機碟會需要 16GB 的容量
所以我買了 32GB 的 USB 碟
想說這樣很夠用

然後到了導引的部分建議我 USB 要大於 32GB…
看了有點崩潰
不過我還是試著先用 32GB 的 USB 來製作
好在製作成功了
實際容量並沒有那麼大

現在很多資訊都會隨時代變化
上網查是一回事
實際遇到的情況又是一回事

拆機更換零件

我的筆電是 Acer Aspire A715-75G
背板會有 13 顆螺絲
卸除螺絲後
慢慢撬開背板

註：3面撬開就可以卸下背板了

第一件事是先拆除電池以防安全
這一步驟是網上學來的

不過因為忘記了這件事

我的操作順序是： 先加裝記憶體 -> 啊！電池好像要先拆 -> 風散看起來要清 -> 換 SSD

風扇區積了很多灰塵
雖然不知道怎清
也沒特別做功課
但還是決定要清

註：過熱也會影響效能，所以散熱很重要

我是用針去挑棉絮狀的灰塵
其他就不管了
因為也不是很懂硬體
在沒做充足功課下就先別亂動

最後來看一下都換好的樣子吧：

使用 USB 開機碟重新安裝作業系統

基本上把硬體的零件都換好後，
重新接上電池和電源，
插入USB開機碟並開機，
就會進到重設電腦的指引會面了。

註：如果沒有的話，可以按F12調整開機時啟用BIOS和USB的順序

話說現在安裝 Windows 11 有一點滿機車的
就是一定要登入微軟帳號…

不過上網查有小技巧可以跳過
那就是一開始選擇連網時先略過
這樣就可以本機直接登入

或是直接輸入錯誤的帳密來登入
這樣也可以回到本機登入

OS: 何必呢…

分割及合併硬碟

一切準備就緒，
我就打開工作管理員和本機，
來檢查是不是硬體都安裝到位了。

記憶體沒什麼問題，
原本就有8GB，
再加裝後就有16GB了！

另外SSD 總共有1TB的容量，
容量是沒問題，
可是硬碟分割的槽卻很瞎。

C槽 700多GB
D槽 200多GB

因為C槽是系統槽，
我偏向是不安裝東西的，
所有遊戲和軟體都安裝在D槽，
這樣心裡感覺比較舒服

所以我想把C槽的部分容量移給D槽，

我一開始是使用內建的硬碟工具，
但遇到了一件更瞎的事，

讓我認識了相鄰與不相鄰的硬碟分割區

這是非常奇葩的規則
如果要合併硬碟上兩個未配置的空間
它們必須相鄰

那我遇上的問題就是
C分割區和D分割區中間卡一個修復磁碟區

上網查了一下
據說可以刪掉
不影響使用但不能再使用修復磁碟機和系統還原的功能
(註：就是製作USB開機碟會用到的工具，但其實已經製作過就不是那麼影響，應該吧…)

另一個分法是使用三方軟體來移動磁碟的分割區
後來我選這個 呵呵

我用的是 AOMIE 免費版本
在一番操作下終於成功把C槽大部分的空間移到D槽了 (撒花

最後就是漫長的重新安裝需要用的軟體(柏德之門3 etc.)的過程哩…
