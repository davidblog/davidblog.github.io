筆電與程式開發

發表於 2023-07-31 | 分類於 學習歷程 | |

程式開發的筆電究竟需要什麼樣的規格？
或許每個人心中都有不同的答案
還有不同的預算 (￣▽￣)ノ

底下就來聊聊幾款我開發用過的筆電：

  • Lenovo IdeaPad 110S - 11 吋
  • Macbook Air i5 2013 - 11 吋
  • Macbook Pro i5 2017 - 13 吋
  • Macbook Air M1 2021 - 13 吋

Lenovo IdeaPad 110S - 11 吋

規格

CPURAMStorageOS
Intel Pentium N3710 1.6GHz4G DDR3L64G EMMCWindows 10 Home

使用緣由

Lenovo 是便宜的選擇當中規格最好的
所以完全是年輕沒錢又要買筆電時的最佳首選
就這麼的單純

這款筆電當初是從燦坤買的
只花了 NTD 9,999
據說是燦坤和 Lenovo 訂製的特殊規格
所以同樣型號的實際規格會比網上查到的還好一些

使用經歷

仔細想想這樣規格的筆電搞開發會有什麼問題嗎？
我當初有用它來開發一款電子旅遊商務網站

專案裡頭沒有使用到任何現代框架
走一個前後端分離的傳統開發路線

基本上只有使用編輯器： Visual Studio Code
還有 Preview on Web Server 的插件來開發
雖然看上去非常克難
但完全夠用

因為開發過程中其實只需要編輯代碼還有預覽畫面就可以了

使用心得

我用過這個牌子的筆電總共有兩台
只聊其中一台是因為剛好只有用一台開發過

那 Lenovo 的筆電我推薦嗎？
嗯… 除非和我當初一樣年輕沒錢…
不然… 不推薦！！！

因為我用過的兩台都有一個通病
就是一年半載後硬體的某個部件就會出問題…

印象中一台是顯示器出問題
一台是鍵盤模組出問題

所以這牌子的品質真的是有待加強…

Macbook Air i5 2013 - 11 吋

規格

CPURAMStorageOS
1.7GHz Intel Core i54GB DDR3128GB SSDmacOS Mojave 10.14.4

使用緣由

專案上的貴人贈與！
剛好我也很想試 macOS 就收了！

使用經歷

這款筆電一樣是用來開發電子旅遊商務網站
但在開發上替換了原本的 Lenovo IdeaPad 110S

使用心得

第一次在 macOS 的使用上
就可以感受到一些明顯的優點
像是日常不用關機
熟悉觸控板可以取代滑鼠
系統使用上比較順暢
畫面色彩佳
硬體壽命長！

當然需要注意的就是：
操作 macOS 和 Windows 有滿多地方不太一樣
所以一開始切換會需要花一些時間熟悉
才能盡情享受使用 macOS 開發的好處

Macbook Pro i5 2017 - 13 吋

規格

CPURAMStorageOS
2.3GHz Intel Core i516GB256G SSD-

使用緣由

專案配發

使用經歷

這台除了一般網頁應用的開發
還擔任了手機跨平台混合式應用的開發

手機跨平台混合式應用
雖然平常撰寫的代碼還是在 VS Code 上

但預覽畫面要用到模擬機還會開：

  • Android Studio
  • Xcode

基本上使用 IDE 的模擬機還滿吃效能的
不用開系統監控 聽聽風扇起飛的聲音就可以知道了

使用心得

這台是我第一次使用蝶式鍵盤的筆電
雖然一開始有點不習慣
但習慣之後發現
鍵盤敲打的觸感也滿好的
特別是按鍵清脆的聲音很有節奏感！

另外在開發上如果有要開比較吃效能的軟體
最好再搭配一個筆電架
筆電架起來和直接放桌上的散熱效果差非常多！

Macbook Air M1 2021 - 13 吋

規格

CPURAMStorageOS
Apple M116GB256GB SSDmacOS Ventura 13.2.1

使用緣由

被媒體燒到
覺得 M1 太屌了 一定要買這樣！

Apple M1 系列是蘋果公司第一款基於 ARM 架構的自研處理器單晶片系統，
於 2020 年 11 月 10 日推出。

M1 是 Apple 的獨家晶片
雖然效能很強
但因為新
所以並不是所有軟體都第一時間相容
我那時還是忍了一陣子觀望才買 XD

買之前一直在逛這個網站 能在 Apple 晶片上使用了嗎？

另外也得知了一個 CPU 跑分網站 PassMark Software

最後買了 16GB 規格的 Macbook Air
價格是 NTD 36,900
買的時間應該落在是 M1 出來過了半年左右的時間 (2021 年 5 月)

話說如果是 8GB 的統一規格只要 NTD 30,900
以蘋果產品的定位來說價格是非常的親民

使用經歷

主要都拿來開發現代化框架的網頁應用居多

使用心得

筆電沒有風散但散熱良好
寂靜運行使用順暢
續航時間長

我認為筆電的重要價值之一就是要能夠隨身攜帶
Air 的輕薄定位剛好就在這裡
然後搭配 M1 的晶片比起過往的 Pro 又有更強的效能

讚！
